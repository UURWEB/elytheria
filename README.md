# Kingdom of Elytheria Tourism Website

A fully static GitHub Pages website for the Kingdom/State of Elytheria in the U.U.R.

## Upload

Upload every file and folder in this directory to the root of a GitHub repository. Enable GitHub Pages from the main branch and root folder.

No Node.js, database, Firebase, or real Java applet is required. The Java-looking features are simulated with HTML, CSS, and JavaScript.

The public site contains no calendar dates.

The exact owner-supplied sunflower flag is stored at `assets/elytheria_flag.webp`. The public palette uses restrained sunflower, earth, olive, and cream tones.
